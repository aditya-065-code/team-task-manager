# Team Task Manager

A responsive full-stack task management web application built using HTML, CSS, JavaScript, Node.js, and Express.js.

This project was developed as part of a Full-Stack Developer assessment to demonstrate frontend development, backend API integration, dashboard UI creation, and responsive web design.

---

# Features

## Authentication
- User Login System
- Session Handling using Local Storage
- Protected Dashboard Access

## Dashboard
- Responsive Dashboard UI
- Sidebar Navigation
- Task Analytics Chart
- Task Status Overview

## Task Management
- Create and Track Tasks
- Task Status:
  - Completed
  - Pending
  - In Progress

## Responsive Design
- Mobile Responsive Layout
- Responsive Sidebar
- Responsive Dashboard Cards

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/your-username/team-task-manager.git
```

---

# Backend Setup

Navigate to backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Run server:

```bash
node server.js
```

Backend will run on:

```bash
http://localhost:5000
```

---

# Frontend Setup

Open frontend folder using Live Server in VS Code.

Or directly run:

```bash
frontend/index.html
```
